La práctica es realmente muy sencilla. Si tienes duda puedes consultar la presentación en:  http://lmrarturo.github.io/GitPresentacion

1)Haz un fork del siguiente repositorio: https://github.com/lmrarturo/FinalGit
2)Agrega a tu repositorio fork a 2 personas más como colaboradores.
3)Pónganse de acuerdo entre los 3 y realicen los siguientes cambios en el proyecto.
 a) Modificar el index.html
 b) Darle estilos al proyecto desde master.css
 c) Crear un nuevo archivo .html y agregarle master.css
 d) Agreguen un javascript sencillo que haga lo que ustedes gusten
4) Todo debe estar en tu repositorio Fork en GitHub
5) Envíame un Pull Request a mi repositorio original.
6) El mejor diseño del proyecto y que tenga buen uso de Git lo pondré como principal en mi repositorio (Aceptaré el Pull Request)

Nota: Si tienes dudas mándenme un tweet a @lmrarturo o un correo a hello@arturomendoza.me
